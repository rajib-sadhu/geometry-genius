What is DOM and what is the purpose of DOM?

DOM means Document Object Model. It's a programming interface for HTML elements, when we start the webpage browser create DOM and we can access all the elements in HTML file through Javascript programme.

DOM is a standard set by W3C. It's mainly life on a web page, user can experience the website works. DOM can life on each and every elements, there are so many properties - click, mousehover, type etc. , every event can call through DOM.

How will you select HTML elements using DOM? Name the DOM methods.

There are many way to access the HTML elements. we can access by tag names, elements id name, elements class names and elements name. two way of methods NodeList and HTMLcollection. Some DOM method given below-

Dom Methods

  • 1. getElementsByTagName()

    This method returns all elements with specified HTML tag name, and it's return HTMLCollection.

  • 2. getElementsByClassName()

    This method returns all elements with specified class names, and it's return HTMLCollection.

  • 3. getElementById()

    This method returns elements with specified id name or value, and it's return HTMLCollection.

  • 4. getElementsByName()

    This method returns elements with specified name, and it's return HTMLCollection.

  • 5. querySelector()

    This method returns first elements which match with CSS selector, and it's return NodeList.

  • 6. querySelectorAll()

    This method returns all elements which match with CSS selector(s), and it's return NodeList.

What is a callback function and why will you use it?

A callback is a function that is passed as an argument to another function, and is called after the main function has finished its execution.

Callbacks allow you to continue executing code while the operation is being executed in the background. Once the operation has completed, the callback function is called with the result of the operation. This way, you can ensure that the program remains responsive and the user experience is not impacted.
